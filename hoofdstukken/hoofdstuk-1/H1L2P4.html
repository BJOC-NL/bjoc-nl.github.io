Namen van spelers leren

Op deze pagina, ga je een groet speler script maken dat gebruikers begroet en hun namen onthoudt zodat nieuwe bezoekers een andere begroeting krijgen dan terugkerende bezoekers.

     

    De keuze wordt gemaakt door een als-anders.

    De als en als-anders blokken worden conditionelen genoemd omdat ze de code besturen aan de hand van een voorwaarde. Hier is die voorwaarde of de naam die de speler geeft in spelerslijst staat.

    Kiezen (selecteren) welk gedeelte van een algoritme uit gevoerd moet worden gebaseerd op een voorwaarde, wordt een selectie genoemd.

    Rangschikken is het process van het ordenen van de code (het in een volgorde plaatsen), zoals je dat hier doet.


  1. Het bevat de meeste code die je nodig hebt om het script te maken.
  2. Voeg de blokken samen in een volgorde zodat:
    1. De groeter zal vragen om de speler zijn/haar naam.
    2. Als deze naam in spelerslijst staat,
      • Zeg "leuk je weer te zien, naam speler."
    3. Anders, wanneer de speler niet op de lijst staat,
      • Zeg "leuk je te ontmoeten, naam speler," en
      • Voeg de naam toe aan spelerslijst.
  3. Test en debug. Zorg ervoor dat het script werkt op de manier waarop je wilt dat het werkt.
  4. Maak een command-blok, en sleep je script hierin.
  5. Klik meerdere keren op groet speler om te testen of het naar behoren werkt. Fix alle bugs die je eventueel nog tegenkomt.
