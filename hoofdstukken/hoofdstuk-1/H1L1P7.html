Het delen van je game

Op deze pagina leer je hoe je je game kunt delen via een link en kunt openen op een mobiele telefoon.
  1. Zorg ervoor dat je project opgeslagen is.
  2. Kies 'Open...' in het bestandsmenu en selecteer het bestand uit je projectlijst. In deze les: 'H1L1-Alonzo'. Klik 'Delen/Share', zie de video hieronder.
    3. Om bestanden snel te vinden in een lange lijst, typ je (een deel van) de naam (bv H1) in de zoekbalk van dit venster. Zie afbeelding.
  3. Er wordt eerst gevraagd of je zeker weet dat je het wilt publiceren (Are you sure...). Klik op 'Ja'. Kies daarna 'Open' of'Annuleren' om het menu te sluiten, welke van de twee maakt niet uit.
  4. Kopieer de URL (het webadres) bovenaan in je browser. Je kan deze link nu naar iemand sturen om je game te delen.
    De link ziet er zo uit: https://snap.berkeley.edu/snap/snap.html#present:Username=...&ProjectName=H1L1-KlikAlonzo.
    Je kan ook een kortere URL maken via Tiny URL om het typen van het webadres simpeler te maken.

How to share a project.
  1. Log uit en laat je partner inloggen. Open de gedeelde link van je game.
    Sla het project ook op in je partners account. Nu heb je allebei een versie van het project.

    Let op! Je hebt zojuist een kopie gemaakt. Een verandering op het ene account is niet automatisch een verandering op het andere account; daarvoor moet je eerst opnieuw de URL van het project delen (en bewaren).
