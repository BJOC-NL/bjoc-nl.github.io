Je kunt pauzeer alles gebruiken, om pauzes in je code te plaatsen zodat je kunt zien wat er gaandeweg gebeurt. In deze animatie, voert de programmeur de code uit met pauzeer alles erin. De programmeur klikt op de Run/Pauze knop naast de groene vlag ( /) om door de code heen te bewegen tot een pauzeblok weer is bereikt. In dit voorbeeld, runt er maar één script, maar pauzeer alles zal elk script pauzeren dat bezig is totdat de gele Run/Pauze knop wordt ingedrukt.

Probleemdecompositie is het proces van het oplossen van een complex probleem door het probleem op te hakken in stukjes en deze stukjes één voor één op te lossen tot je het originele probleem hebt opgelost.
